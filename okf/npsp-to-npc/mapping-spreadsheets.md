# NPSP/legacy-package to AFNP mapping spreadsheets

[Appendix A](migration-guide.md) of the migration guide links 4 workbooks
total -- each mapping one legacy managed package's fields to their
Agentforce Nonprofit (AFNP) equivalents. Direct download links live on
the (JavaScript-rendered) migration guides help article, not resolvable
by this bundle's own tooling -- visit the article directly to download
any of the three not yet reviewed.

| Workbook | Legacy package | Status |
|---|---|---|
| Nonprofit Success Pack to Agentforce Nonprofit Migration Mappings | NPSP (core) | **Reviewed** -- see [npsp-to-afnp-field-mapping.md](npsp-to-afnp-field-mapping.md) |
| Program Management Module & Case Management to Agentforce Nonprofit Migration Mappings | PMM & NCCM | Not yet reviewed |
| Outbound Funds Module & Grant Management Module to Agentforce Nonprofit Migration Mappings | OBF & GMM | Not yet reviewed |
| Volunteers for Salesforce to Agentforce Nonprofit Migration Mappings | V4S | Not yet reviewed |

(The guide's own Appendix A text names exactly 3 additional workbooks --
confirmed by direct extraction of the PDF's real text, not assumed. Each
name covers a package pairing already bundled together by Salesforce,
e.g. "Program Management Module & Case Management" is one workbook, not
two.)
