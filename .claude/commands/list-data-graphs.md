List the org's Data Graphs.

1. Run: `.venv/Scripts/python.exe cli.py list-data-graphs`
2. Paste the actual output into the reply — don't summarize it.
3. "No Data Graphs found" is a normal answer for an org that hasn't
   configured any — for run/refresh status of ones that do exist, use
   `/data-cloud-status data-graph` instead.

Read-only — safe to run without confirmation.
